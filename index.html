Welcome to the 6Linux homepage!

6Linux is a lightweight, simple and easy to use Linux distribution using the Linux kernel from https://github.com/torvalds/linux and BusyBox 1.36.1. When an official image is released, PLEASE DO NOT USE IT ON YOUR NORMAL MACHINE! Use it in a VM. There will be a download button that will take you to the download page when it launches.
