Welcome to the 6Linux guides!


To use 6Linux:

1. Download the archive

Download the archive from the main web page.


2. Extract the archive

Extract the newly downloaded archive with a tool like WinRAR or 7Zip.


3. Run the QEMU VM

After extracting the archive, run this command in Command Prompt (in the extracted archive directory): "\qemu-system-x86_64.exe boot", where is the directory you installed QEMU in (normally, it is located in C:\Program Files\qemu\).

After opening QEMU with the image, you will see that after "boot:" you will have to type something, type "/bzImage -initrd=/init.cpio", wait for a couple of seconds, and then press ENTER a couple times, and you are using 6Linux.
